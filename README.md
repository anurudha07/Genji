Language: TypeScript
Backend - Node.js, Express.js
Database - MongoDB (Mongoose)
Authentication - JWT, OTP (bcrypt), Google OAuth, Rate Limit
Froentend - React.js, Vite, Tailwind
File Storage - Cloudinary
Scheduler - Node-cron
